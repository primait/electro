# electro

L'unico **electro** senza Electron

## Install

### Clone repository

```
git clone https://github.com/primait/electro.git ~/.electro
```

### Add to your profile file

* Bash

```
echo -e '\n. $HOME/.electro/electro.sh >> ~/.bash_profile
```

* Zsh

```
echo -e '\n. $HOME/.electro/electro.sh >> ~/.zshrc
```
